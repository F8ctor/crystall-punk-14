
# Color Replcaer

## The right version of python: 3.10+

## Settings

### Pallets in - pallets.py

### Main settings in - main.py

## Dependencies Install

in console

```bash
pip install -r requirements.txt
```
or just run

```bash
install_packages.bat
```

## Run
in console

```bash
python main.py
```
or just run

```bash
run.bat
```
